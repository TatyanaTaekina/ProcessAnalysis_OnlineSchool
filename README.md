# Анализ бизнес-процессов онлайн-школы
## Решение проблем с помощью совершенствования процессов

В презентации вы найдете:
  * Описание текущей ситуации и проблем
  * Построение основных диаграмм для анализа
  * Выбор приоритетных процессов для оптимизации
  * Подробное моделирование (включая BPMN) и анализ выбранного процесса (AS IS)
  * Предложения по улучшению процесса и модель TO BE
  * Стратегическую карту BSC

Из-за конфиденциальности, использована несуществующая онлайн-школа, однако описание процессов составлено на основе реальной компании. 

Проект выполнен в рамках учебрной дисциплины "Моделирование и анализ бизнес-процессов" в команде 3 человек (ВШЭ, БИ, 2025)
